GET STARTED

    API Endpoint

        https://aurora-theogenia.de/clever_charge/

The Clever Charge API provides programmatic access to a mysql database. It converts http requests to mysql statements. You can catch the response with a json libary.

To use this API, you need an ACCESS TOKEN. Please contact me at Discord to get a token. Backxtar#3244

REGISTER


# Here is an example
GET https://aurora-theogenia.de/clever_charge/register/
?access_token=your_token
&username=test
&userpasswd=1234
&useremail=test@test.com

To register an account you need to make a GET call to the following url :
https://aurora-theogenia.de/clever_charge/register/ 



Result example :

{
    "response_code": 1,
    "response_msg": "insert success"
}

QUERY PARAMETERS

Field Type Description
access_token String Your token.
username String username for the account to be registered.
userpasswd String password for the account to be registered.
useremail String email for the account to be registered.

?access_token=your_token

https://aurora-theogenia.de/clever_charge/set_fav/

}

REQUEST CODES

The Clever Charge API uses the following request codes:

Request Code Meaning
0 Action failed.
1 Action successful.
2 Some parameters are missing. This error appears when you don't pass every parameters.
69 Your token is not valid.
